¿Qué son?

Business Intelligence tiene como misión principal transformar los datos en información aprovechable para que se puedan tomar decisiones basadas en datos. Dentro del BI, el Data Engineering se dedica a transformar y dar accesibilidad a los datos para que los Data Scientist y los Data Analysts puedan desarrollar su trabajo. Las principales funciones que tienen son

ETL

Generación y mantenimiento de los ETL. Los ETL (Extract Load and Transform) son los procesos en los que se transforman los datos y se cargan en la Data Warehouse desde diferentes fuentes de datos. Estas fuentes pueden ser otras bases de datos, archivos de texto, páginas web, etc.

Este proceso puede ser desarrollado con múltiples herramientas o con código propio. Dependiendo de la casuística a realizar se seleccionará unas u otras. En la actualidad las herramientas de ETL más populares son:

  • Glue - Amazon Web Services es un servicio de integración de datos sin servidor que facilita la detección, preparación, migración e integración de datos provenientes de varios orígenes para el análisis, machine learning (ML) y desarrollo de aplicaciones.
  • Data Factory – Azure, un servicio de integración de datos sin servidor totalmente administrado. Integra visualmente los orígenes de datos con más de 90 conectores integrados que no necesitan mantenimiento, y sin costo adicional. Crea fácilmente procesos de ETL y ELT sin necesidad de programación en un entorno intuitivo.

Data Warehouse

Un data warehouse es un repositorio unificado para todos los datos que recogen los diversos sistemas de una empresa. El repositorio puede ser físico o lógico y hace hincapié en la captura de datos de diversas fuentes sobre todo para fines analíticos y de acceso.

Dependiendo del tipo de información que se quiera utilizar, se utilizará una base de datos u otra. En un caso estándar, se recurrirá a una bbdd SQL y se utilizará la metodología Kimball para su desarrollo con esquemas de estrella o de copo de nieve.

OLAP y Data MiningSistemas BI más populares

Son los encargados de minar todos los datos disponibles y transformarlos en información (data mining) para la toma de decisiones. Para ello generalmente se utiliza código (Python/R). Para poder visualizar los datos las herramientas más populares y potentes en la actualidad son:

  • Microsoft Power BI.
  • Tableau
  • Microstrategy
  • Looker
  • Qlik
