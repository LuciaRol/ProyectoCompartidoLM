Really Simple Syndication

¿Qué es?

Dialecto XML que se usa para distribuir contenido en la web mediante un sistema de suscripción. Podemos obtener información actualizada de varias web en un solo lugar, llamados agregadores de contenido.

Ejemplos de utilización

Formato Atom

¿Qué es?

Formato para publicar contenido en XML, también se usa para distribuir contenido de agregadores de información mediante suscripción a los distintos canales. La principal diferencia es que Atom se desarrolló para corregir algunos de los fallos de RSS.
